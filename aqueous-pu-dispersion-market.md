<p><strong>The Aqueous PU Dispersion Market Analysis by types is segmented into:</strong></p>
<p><ul><li>Component Polyurethane</li><li>Two-component Polyurethane</li><li>Urethane Modified</li></ul></p>
<p><p>Aqueous PU dispersion market has three main types: Component Polyurethane, Two-component Polyurethane, and Urethane Modified. Component Polyurethane involves a single-component system where the application is ready for use without any mixing required. Two-component Polyurethane consists of two components that need to be mixed together before application to achieve desired properties. Urethane Modified involves adding urethane groups to improve certain properties of the dispersion. Each type offers specific characteristics and advantages for different applications in the market.</p></p>
